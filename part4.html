  9. Explain the types of relationships in database.

    • One-to-One. table A can have many matching rows in table B, but a row in table B can have only one matching row in table A.
    • One-to-Many. a row in table A can have many matching rows in table B, but a row in table B can have only one matching row in table A.
    • Many-to-Many. a row in table A can have many matching rows in table B, and vice versa.
